Tags, Elements, and Attributes, Oh My!

TAGS

The most basic concept of HMTL is a tag. A tag define an HTML element by he "greater than" and "less than" signs. Or, more formally, angled brackets. Like so: < >. The closing tag contains a forward slash: < / >.
Some tags, such as < br > do not require closing brackets and are called "void tags."

ELEMENTS

HTML tags contain elements. This is the foundation of a HTML, and therefore, a web page. This is an example of the "title" element, written out: < title > TITLE < / title >.

ATTRIBUTES

A property or modifier of an element is an attribute . HREF is an attribute that I have used thus far to link to outside websites.

ELEMENT CATAGORIES

Block Level and Inline are element catagories. Block Level elements exist independently, sort of like discrete lines on a piece of binder paper.

Inline elements display "continuously," without discrete seperation. Imagine a line of your binder paper continuing on and on and on....

DIV and SPAN

DIV=Block

1) Think of this as a box!

2) Often used with CSS

3) Example elements include: < div >, < p >, < form >.

SPAN=Inline
